Opdracht 6

Je kan vinden welke usb apparaten verbonden waren met jouw windows computer in de registry editor. Als je dit heb geopend ga je naar HKEY_LOCAL_MACHINE daarna naar SYSTEM, CurrentControlSet , Enum en als laatst USB.

Het kan echter ook makkelijk door het volgende command in een windows powershell uit te voeren:

Get-ItemProperty -Path HKLM:\SYSTEM\CurrentControlSet\Enum\USB\*\* | Select FriendlyName
